1️⃣ Crear y activar un entorno virtual
python -m venv .env
.env\Scripts\activate

2️⃣ Clonar el proyecto desde GitHub
git clone https://github.com/tu_usuario/mi-repo.git
cd mi-repo

3️⃣ Instalar dependencias
Si subiste un requirements.txt, instala todas las librerías necesarias:
pip install -r requirements.txt

Nota: crear una bd(PosgresSQL) con el nombre 'SIREPRE, luego en opciones backup para
elegir el backup de la base de datos que se encuentra en la carpeta del proyecto

4️⃣ Configurar la base de datos
Edita settings.py con los datos correctos (NAME, USER, PASSWORD, HOST, PORT)

5️⃣ Aplicar migraciones (si en caso no se tiene la bd principal)
python manage.py makemigrations
python manage.py migrate

6️⃣ Crear superusuario (opcional)
python manage.py createsuperuser

7️⃣ Levantar el servidor de desarrollo
python manage.py runserver
